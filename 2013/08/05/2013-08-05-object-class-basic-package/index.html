Integer类的概述和构造方法

    • 通过JDK提供的API，查看Integer类的说明

    • Integer 类在对象中包装了一个基本类型 int 的值,

    • 该类提供了多个方法，能在 int 类型和 String 类型之间互相转换，
    • 还提供了处理 int 类型时非常有用的其他一些常量和方法

  • 构造方法
