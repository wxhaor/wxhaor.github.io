循环结构三种循环语句的区别

  • A:代码演示
    • 三种循环语句的区别:
    • do…while循环至少执行一次循环体。
    • 而for,while循环必须先判断条件是否成立，然后决定是否执行循环体语句。
  • for循环和while循环的区别：
    如果你想在循环结束后，继续使用控制条件的那个变量，用while循环，否则用for循环。不知道用谁就用for循环。因为变量及早的从内存中消失，可以提高内存的使用效率。

循环结构注意事项之死循环

  • 一定要注意控制条件语句控制的那个变量的问题，不要弄丢了，否则就容易死循环。
  • 两种最简单的死循环格式
    • while(true){…}
    • for(;;){…}
